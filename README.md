# 🧠 Flashcards

A personal knowledge base built on spaced repetition. The goal is to retain concepts long-term across subjects like math and physics by reviewing them at optimal intervals.

Cards are generated by an autonomous agent and stored as plain Markdown files. The [hashcards](https://github.com/masolnada/hashcards) client runs as part of my [homelab](https://github.com/masolnada/homelab).

## 📁 Structure

Decks are grouped by subject. Each `.md` file is a deck.

```
cards/
├── math/
└── electromagnetism/
```
